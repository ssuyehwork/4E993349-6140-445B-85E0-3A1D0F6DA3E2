# Product Requirements (功能需求全集)

请用 C++ 完美实现以下所有功能：

### 1. 桌面悬浮球 (Hyper Floating Ball)
- **外观**：异形窗口（圆形），支持透明背景，无锯齿。
- **物理交互**：
  - 惯性拖拽：松手后球体带有物理减速效果。
  - **边缘吸附**：拖至屏幕边缘自动半隐藏，鼠标靠近时弹出。
- **功能**：
  - **拖拽投喂**：拖入文本/文件直接保存到数据库。
  - **皮肤切换**：右键菜单可切换皮肤（摩卡·勃艮第、经典黑金、皇家蓝、抹茶绿）。
  - **粒子特效**：记录成功时，球体周围迸发粒子动画（使用 `QPainter` 高性能绘制）。
  - **双击**：快速唤醒“快速记录窗口”。

### 2. 快速记录窗口 (Spotlight Quick Window)
- **定位**：类似 uTools/Spotlight 的居中无边框窗口，按 Alt+Space 秒开。
- **布局**：
  - **顶部**：搜索栏（支持历史记录回溯）。
  - **左侧**：笔记列表（支持 Ctrl/Shift 多选、拖拽排序）。
  - **右侧**：侧边栏（系统分类：全部、今日、未分类、回收站；用户自定义分类树）。
  - **工具栏**：垂直工具栏（置顶、翻页、最小化）。
- **特性**：支持键盘上下键快速选择，Enter 键复制内容。

### 3. 主管理界面 (Pro Main Window)
- **布局**：专业的三栏式设计（侧边栏树状图 | 中间卡片列表 | 右侧编辑器/详情）。
- **中间列表**：
  - 卡片式设计，显示标题、预览摘要（3行）、创建时间、标签气泡、星级评分。
  - 状态图标：置顶（图钉）、锁定（锁）、收藏（书签）。
- **右侧元数据面板**：可折叠，用于修改标题、管理标签（Tag）、设置颜色。

### 4. 增强型编辑器 (Advanced Editor)
- **Markdown 支持**：基于正则表达式的实时语法高亮（标题变色、加粗、列表）。
- **双向链接**：输入 `[[` 自动弹出笔记列表，回车插入链接，点击跳转。
- **智能粘贴**：粘贴图片自动保存到数据库 Blob，粘贴文件路径自动识别。

### 5. 知识可视化 (New Feature: Graph View)
- **知识图谱**：使用 `QGraphicsView` 框架，将所有笔记渲染为节点，引用关系渲染为连线。
- **物理引擎**：实现力导向算法（Force-directed graph），节点之间有斥力，连线有拉力，支持鼠标拖拽节点产生弹性动画。

### 6. 实用工具箱 (Toolbox)
- **屏幕截图与 OCR**：集成截图工具，截图后自动调用 Windows API (OCR) 识别文字并存入笔记。
- **密码生成器**：生成高强度密码（含大小写、符号）。
- **时间戳助手**：快速复制当前时间/日期。

### 7. 底层核心系统
- **数据库**：SQLite3，启用 FTS5 全文搜索，支持毫秒级检索。
- **剪贴板监听**：后台静默监听，内容去重（SHA256 哈希比对），变化时托盘气泡提示。
- **全局热键**：注册系统级热键（Alt+Space, Ctrl+Shift+E）。
